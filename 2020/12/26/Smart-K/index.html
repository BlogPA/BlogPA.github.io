逻辑运算符

运算符 含义 效果
&& 逻辑与运算符 连接表达式，两个表达式必须都为 true，整个表达式才为 true
|| 逻辑或运算符 连接表达式，只要其中有一个表达式为 true，则结果就为 true
| 逻辑非运算符 反转表达式的结果，使 ture 变成 flase 或者 flase 变成 true

访问说明符的定义

Public：意味着对所有人都可用

Private：只对类型的创建者和类内部成员函数可用

Protected：与 private 类似，只有一点不同，即继承的类可以访问 protected 成员，但不能访问private成员

C++ 的基本输入输出

I / O 库头文件，输入输出需要声明 #include <iostream>

  • 标准输出流 （cout）：预定义的对象 cout 是 iostream 类的一个实例。cout 对象”连接”到标准输出设备，通常是显示屏。cout 是与流插入运算符 << 结合使用。
  • 标准输入流 （cin）：预定义的对象 cin 是 iostream 类的一个实例。cin 对象附属到标准输入设备，通常是键盘。cin 是与流提取运算符 >> 结合使用。
